# underlog
Log viewer

## Features
- log stepping
- log viewing for huge files
- highlitng rules
- grouping rules
- categories and filers
- get stats
- detect loops
- export to SQL and GraphDB formats
- merge logs
- TUI view
- GUI in browser view
